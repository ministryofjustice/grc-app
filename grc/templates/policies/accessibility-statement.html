  • {{ _('zoom in up to 300%% without problems') }}
  • {{ _('navigate most of the website using just a keyboard') }}
  • {{ _('navigate most of the website using speech recognition software') }}
  • {{ _('use most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)') }}
